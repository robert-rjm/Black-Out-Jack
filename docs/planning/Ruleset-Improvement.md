# Improved Ruleset for beginners / extended comprehensive example

**aim:** Easily onboard new players with simplified Rules and provide better/ up to date comprehensive example illustrating these rules (potentially link rules directly to example).

Applies only to the **Drinking Game mode**, as these are the rules to be explained that differentiate from standardized BlackJack rules.


## Specifically new features:
- improved `docs/Comprehensive_Example.md` file including all up to date rules
    - potentially linked to `docs/Rules.md` for easy comparison and illustration
- simplified rules to onboard new players
    - exclude complex rules and allow "5-min onboarding"
    - potentially give admin options to select/ unselect specific rules (allow admin to control continous between "Simplifed" rules and full rules)
